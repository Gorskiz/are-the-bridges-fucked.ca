Welcome to the *unofficial*, *unfiltered*, and *undeniably better* way to check if the Macdonald or MacKay bridges are currently a parking lot.

## Why? 
Because the [official site](https://halifaxharbourbridges.ca/) has the charm of a 1998 Geocities page and the user experience of a tax audit. Haligonians deserve better. We deserve to know if we're going to be late for work *in style*.

## What is this?
A bespoke, high-performance web application dedicated to answering the age-old Halifax question:  
**"Can I get across the harbour, or should I just go home?"**
